# Descrição

Você tem dois amigos que estão conversando em Pig Latin! Pig Latin é uma forma modificada das mesmas palavras, na mesma ordem, mas com a regra de que a primeira letra de cada palavra é movida para o final da palavra, e então adiciona-se 'ay' ao final. Por exemplo, "road" se torna "oadray".

## Tarefa

Sua tarefa é pegar uma frase em inglês e transformá-la na mesma frase em Pig Latin!

## Formato de Entrada

Uma string contendo uma frase em inglês que você precisa traduzir para Pig Latin. (sem pontuação ou letras maiúsculas).

## Formato de Saída

Uma string contendo a mesma frase em Pig Latin.

## Exemplo de Entrada

nevermind youve got them

## Exemplo de Saída

evermindnay ouveyay otgay hemtay

## Explicação

- A palavra "nevermind" se torna "evermindnay" em Pig Latin: a primeira letra "n" é movida para o final, e "ay" é adicionado.
- A palavra "youve" se torna "ouveyay".
- A palavra "got" se torna "otgay".
- A palavra "them" se torna "hemtay".
- O programa deve aplicar essa regra a todas as palavras da frase, mantendo-as na mesma ordem e retornando a frase completa em Pig Latin.

Este arquivo .md descreve claramente o problema, o formato de entrada e saída, e oferece uma explicação passo a passo para a transformação de cada palavra em Pig Latin.
